# Submission Guide
1. **Fork** this repository.
2. **Clone** your fork to your machine.
3. Train your model using `data/train.csv`.
4. Run your model on `data/test_features.csv` to create a file named `yourteam_submission.csv`.
   - Format: A single column named `Outcome` with your 0/1 predictions.
5. Place this CSV in the `submissions/` folder.
6. **Commit and Push** to your fork.
7. Open a **Pull Request (PR)** from your fork to the main repo's `main` branch.
8. Once the instructor merges your PR, the leaderboard will update automatically!
